## Теоретическая справка

### Историческая справка

Первые упоминания о матрицах (или как их тогда называли — «волшебных квадратах») были обнаружены на территории Древнего Китая.

Свою известность они получили в середине XVIII века благодаря труду знаменитого математика Габриэля Крамера, опубликовавшего свой труд «Введение в анализ алгебраических кривых», в котором описывался принципиально новый алгоритм решения систем линейных уравнений.

Вскоре после него были опубликованы работы Карла Фридриха Гаусса о «классическом» методе решения линейных уравнений, теорема Гамильтона‑Кели, работы Карла Вейерштрасса, Георга Фробениуса и других выдающихся ученых.

И только в 1850 году Джеймс Джозеф Сильвестр в своей работе вводит термин «Матрица».

### Матрица

Матрица — это набор чисел, расположенных в фиксированном количестве строк и столбцов.

Матрица A — прямоугольная таблица чисел, расположенных в m строках и n столбцах:

```
1 2 3
A = 4 5 6
7 8 9
```

Получить нужный элемент можно при помощи индексов: `A[0][0] = 1`, где первый индекс — номер строки, второй — номер столбца.

**Порядок матрицы** — число ее строк или столбцов (для квадратной матрицы).

**Главная диагональ** квадратной матрицы — диагональ, идущая из левого верхнего в правый нижний угол.

**Квадратная матрица** — число строк равно числу столбцов.

**Прямоугольная матрица** — число строк не равно числу столбцов.

**Матрица-столбец** — матрица с одним столбцом.

**Матрица-строка** — матрица с одной строкой.

**Диагональная матрица** — квадратная матрица, у которой все элементы вне главной диагонали равны нулю.

**Единичная матрица** — диагональная матрица с единицами на главной диагонали.

### Структура матрицы на языке C:

```c
typedef struct matrix_struct {
    int** matrix;
    int rows;
    int columns;
} matrix_t;
```

### Библиотека `matrix`

Библиотека должна содержать следующие обязательные функции:

### Программа `main.c`

Файл `main.c` содержит демонстрацию работы библиотеки:

- создаёт матрицы через `create_matrix` и заполняет их значениями;

**Запрещается** использование стандартных функций из `<stdio.h>`, `<stdlib.h>`, `<string.h>` и любых других, не входящих в libft (кроме разрешённых системных вызовов `write`, `read`, `malloc`, `free`, которые могут использоваться только внутри libft и matrix).

### Общие требования ко всем вариантам

### Вариант 1. Сложение и вычитание двух матриц

**Теория:**

Суммой двух матриц A = m × n и B = m × n одинаковых размеров называется матрица C = m × n = A + B тех же размеров, элементы которой определяются равенствами:

```
C(i,j) = A(i,j) + B(i,j)
```

Разностью двух матриц A = m × n и B = m × n одинаковых размеров называется матрица C = m × n = A − B тех же размеров, элементы которой определяются равенствами:

```
C(i,j) = A(i,j) − B(i,j)
```

Пример:

```
            1 2 3   1 0 0   2 2 3
C = A + B = 0 4 5 + 2 0 0 = 2 4 5
            0 0 6   3 4 1   3 4 7
```

**Описание:**

Реализуйте функции сложения и вычитания двух матриц. В `main.c` создайте две матрицы одинаковых размеров, заполните их, выполните обе операции и выведите результаты.

**Функции для реализации:**

```c
matrix_t *sum_matrix(matrix_t *A, matrix_t *B);
matrix_t *sub_matrix(matrix_t *A, matrix_t *B);
```

**Пример использования в `main.c`:**

```c
matrix_t *A = create_matrix(2, 2);
A->matrix[0][0] = 1; A->matrix[0][1] = 2;
A->matrix[1][0] = 3; A->matrix[1][1] = 4;

matrix_t *B = create_matrix(2, 2);
B->matrix[0][0] = 5; B->matrix[0][1] = 6;
B->matrix[1][0] = 7; B->matrix[1][1] = 8;

matrix_t *C = sum_matrix(A, B);
print_matrix(C);
// Результат:
// 6 8
// 10 12

remove_matrix(A);
remove_matrix(B);
remove_matrix(C);
```

### Вариант 2. Умножение матрицы на число и транспонирование

**Теория:**

Произведением матрицы A = m × n на число λ называется матрица B = m × n = λ × A, элементы которой определяются равенствами:

```
B(i,j) = λ × A(i,j)
```

Пример:

```
            1 2 3   2 4 6
B = 2 × A = 0 4 2 = 0 8 4
            2 3 4   4 6 8
```

Транспонирование матрицы A заключается в замене строк этой матрицы её столбцами с сохранением их номеров:

```
          1 4   1 2 3
A = A^T = 2 5 = 4 5 6
          3 6
```

**Описание:**

Реализуйте функции умножения матрицы на число и транспонирования. В `main.c` создайте матрицу, выполните обе операции и выведите результаты.

**Функции для реализации:**

```c
matrix_t *mult_number(matrix_t *A, int num);
matrix_t *transpose(matrix_t *A);
```

**Пример использования в `main.c`:**

```c
matrix_t *A = create_matrix(2, 3);
A->matrix[0][0] = 1; A->matrix[0][1] = 2; A->matrix[0][2] = 3;
A->matrix[1][0] = 4; A->matrix[1][1] = 5; A->matrix[1][2] = 6;
```

### Вариант 3. Умножение двух матриц

**Теория:**

Произведением матрицы A = m × k на матрицу B = k × n называется матрица C = m × n = A × B размера m × n, элементы которой определяются равенством:

```
C(i,j) = A(i,1) × B(1,j) + A(i,2) × B(2,j) + … + A(i,k) × B(k,j)
```

Пример:

```
            1 4    1 -1  1    9 11 17
C = A × B = 2 5  × 2  3  4 = 12 13 22
            3 6              15 15 27
```

Компоненты матрицы C вычисляются следующим образом:

```
C(1,1) = A(1,1) × B(1,1) + A(1,2) × B(2,1) = 1 × 1 + 4 × 2 = 9
C(1,2) = A(1,1) × B(1,2) + A(1,2) × B(2,2) = 1 × (-1) + 4 × 3 = 11
C(1,3) = A(1,1) × B(1,3) + A(1,2) × B(2,3) = 1 × 1 + 4 × 4 = 17
C(2,1) = A(2,1) × B(1,1) + A(2,2) × B(2,1) = 2 × 1 + 5 × 2 = 12
C(2,2) = A(2,1) × B(1,2) + A(2,2) × B(2,2) = 2 × (-1) + 5 × 3 = 13
C(2,3) = A(2,1) × B(1,3) + A(2,2) × B(2,3) = 2 × 1 + 5 × 4 = 22
C(3,1) = A(3,1) × B(1,1) + A(3,2) × B(2,1) = 3 × 1 + 6 × 2 = 15
C(3,2) = A(3,1) × B(1,2) + A(3,2) × B(2,2) = 3 × (-1) + 6 × 3 = 15
C(3,3) = A(3,1) × B(1,3) + A(3,2) × B(2,3) = 3 × 1 + 6 × 4 = 27
```

**Описание:**

Реализуйте функцию умножения двух матриц. В `main.c` создайте две совместимые матрицы, вычислите произведение и выведите результат.

**Функция для реализации:**


Функция возвращает `NULL`, если `A->columns != B->rows`.

**Пример использования в `main.c`:**

```c

A->matrix[0][0] = 1; A->matrix[0][1] = 2; A->matrix[0][2] = 3;
A->matrix[1][0] = 4; A->matrix[1][1] = 5; A->matrix[1][2] = 6;
```

### Вариант 4. Сравнение матриц и анализ свойств

**Теория:**

**Сравнение матриц** — две матрицы A и B равны, если они имеют одинаковые размеры и все соответствующие элементы равны:

```
A == B  ⇔  A.rows == B.rows  &&  A.cols == B.cols  &&  A(i,j) == B(i,j)  ∀ i,j
```

**Симметричная матрица** — квадратная матрица, равная своей транспонированной:

```
A = Aᵀ  ⇔  A(i,j) == A(j,i)  ∀ i,j
```

Пример симметричной матрицы:

```
    1 2 3
A = 2 5 6
    3 6 9
```

**Диагональная матрица** — квадратная матрица, у которой все элементы вне главной диагонали равны нулю:

```
A(i,j) == 0  при  i != j
```

Пример:

```
    1 0 0
A = 0 5 0
    0 0 9
```

**Единичная матрица** — диагональная матрица с единицами на главной диагонали:

```
A(i,j) == 1  при  i == j
A(i,j) == 0  при  i != j
```

Пример:

```
    1 0 0
E = 0 1 0
    0 0 1
```

**Описание:**

Реализуйте функции сравнения двух матриц и проверки свойств квадратной матрицы. В `main.c` создайте матрицы, проверьте их свойства и выведите `1` (да) или `0` (нет).

**Функции для реализации:**

```c
int eq_matrix(matrix_t *A, matrix_t *B);
int is_symmetric(matrix_t *A);
int is_diagonal(matrix_t *A);
int is_identity(matrix_t *A);
```

Каждая функция возвращает `1`, если условие выполняется, и `0` в противном случае.

**Пример использования в `main.c`:**

```c
matrix_t *A = create_matrix(2, 2);
A->matrix[0][0] = 1; A->matrix[0][1] = 2;
A->matrix[1][0] = 3; A->matrix[1][1] = 4;

matrix_t *B = create_matrix(2, 2);
B->matrix[0][0] = 1; B->matrix[0][1] = 2;
B->matrix[1][0] = 3; B->matrix[1][1] = 4;

ft_putnbr_fd(eq_matrix(A, B), 1);  // 1
ft_putchar_fd('\n', 1);

matrix_t *C = create_matrix(3, 3);
C->matrix[0][0] = 1; C->matrix[0][1] = 2; C->matrix[0][2] = 3;
C->matrix[1][0] = 2; C->matrix[1][1] = 5; C->matrix[1][2] = 6;
C->matrix[2][0] = 3; C->matrix[2][1] = 6; C->matrix[2][2] = 9;

ft_putnbr_fd(is_symmetric(C), 1);  // 1
ft_putchar_fd('\n', 1);

remove_matrix(A);
remove_matrix(B);
remove_matrix(C);
```

### Makefile

В Makefile должны быть цели:

- `all` — сборка программы;
- `clean` — удаление объектных файлов;
- `fclean` — удаление объектных файлов и исполняемого файла;
- `re` — пересборка.

**Пример Makefile:**

```makefile
NAME = matrix
CC = gcc
CFLAGS = -Wall -Wextra -Werror -I./inc
LIBFT_DIR = ./libft
LIBFT = $(LIBFT_DIR)/libft.a

SRC_DIR = src
SRCS = $(SRC_DIR)/main.c \
       $(SRC_DIR)/create_matrix.c \
       $(SRC_DIR)/remove_matrix.c \
       $(SRC_DIR)/print_matrix.c \
       $(SRC_DIR)/sum_matrix.c \
       $(SRC_DIR)/sub_matrix.c
OBJS = $(SRCS:.c=.o)

all: $(NAME)

$(NAME): $(OBJS) $(LIBFT)
	$(CC) $(CFLAGS) $^ -o $@

$(LIBFT):
	make -C $(LIBFT_DIR)

%.o: %.c
	$(CC) $(CFLAGS) -c $< -o $@

clean:
	rm -f $(OBJS)
	make -C $(LIBFT_DIR) clean

fclean: clean
	rm -f $(NAME)
	make -C $(LIBFT_DIR) fclean

re: fclean all

.PHONY: all clean fclean re
```

> **Примечание:** в примере Makefile указаны файлы для варианта 1. Для других вариантов замените `sum_matrix.c`, `sub_matrix.c` на соответствующие файлы вашего варианта.
